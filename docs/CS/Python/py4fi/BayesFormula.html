Bayesian Regression¶

Bayes' Formula¶

We learn new information about certain variables or parameters of interest, like the mean return of a time series. $$ p(H|D)=\dfrac{p(H)p(D|H)}{p(D)}$$ p(H) is called prior probability. p(D) is the probability for the data under any hypothesis, called the normalizing constant. P(D|H) is the likehood of the data under hypothesis H. P(H|D) is the posterior probability after we have seen the data.

Example:¶

We have two boxes, B1 and B2 , B1 contains 20 black balls and 80 white balls, B2 contains 30 black balls and 70 white balls. We randomly select a ball from one box, assuming it's black. What's the probability for the hypothesis "H1: ball is from B1" and "H2: ball is from B2".

  • Prior: p(H1) = 0.5
  • Normalizing constant: p(D) = 0.20.5 + 0.30.5 = 0.25
  • Likehood: p(D|H1) = 0.2 Then $$p(H1|D) = \dfrac{0.5*0.2}{0.25}=0.4$$

Bayesian regression¶

line $y=\alpha+\beta*x$

  • $\alpha$ and $\beta$ are normally distributed with mean 0 and a standard deviation of 20. A majority element of Bayesian regression is (Markov Chain) Monte Carlo (MCMC) sampling.
  • find_MAP finds the starting point for the sampling algorithm by deriving the cal maximum a posteriori point.
  • NUTS implements the "efficient No-U-Turn Sampler with dual averaging" (NUTS) algorithm for MCMC sampling given the assumed priors.
  • sample draws a number of samples given the starting value from find_MAP and the optimal step size from the NUTS algorithm.

from pymc3.distributions.timeseries import GaussianRandomWalk
# to make the model more simple, we will apply the same coefficients
# to 50 data points at a time
subsample_alpha = 50
subsample_beta = 50

with model_randomwalk:
    alpha = GaussianRandomWalk('alpha', sigma_alpha**-2, 
                               shape=len(data) / subsample_alpha)
    beta = GaussianRandomWalk('beta', sigma_beta**-2, 
                              shape=len(data) / subsample_beta)
    
    # make coefficients have the same length as prices
    alpha_r = np.repeat(alpha, subsample_alpha)
    beta_r = np.repeat(beta, subsample_beta)
